Download

Following are downloads for the latest version of Project: Starfighter. I cannot build Starfighter for all systems, so if your OS is not listed here and you would like it to be, feel free to send a build to me at diligentcircle@riseup.net. If you would like an older version of Starfighter, you can find it in the list of releases on GitHub or the old Savannah releases directory.

  • Linux (64-bit)
  • Windows
  • Source code (tar.gz)
  • Source code (zip)

If you'd like to see how far the game has come, or if you'd simply like to see how the original game looked (with its unlicensed art assets), the Wayback Machine still has a copy of version 1.1, the last version released by Parallel Realities, archived. Do keep in mind that this old code is filled with bugs, including memory access bugs, albeit memory access bugs that shouldn't cause problems outside the game. It can be found here: https://web.archive.org/web/20051016190326/http://www.parallelrealities.co.uk/download.php?file=starfighter-1.1-1.tar.gz&type=zip
