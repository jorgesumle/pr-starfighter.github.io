Download Project: Starfighter

Following are downloads for the latest version of Project: Starfighter. I cannot build Starfighter for all systems, so if your OS is not listed here and you would like it to be, feel free to send a build to me at onpon4@riseup.net. If you would like an older version of Starfighter, you can find it in the releases directory.

  • Windows: starfighter-1.7-win32.zip (Alternate link)
  • GNU/Linux (32-bit): starfighter-1.7-gnulinux-i486.tar.gz (Alternate link)
  • GNU/Linux (64-bit): starfighter-1.7-gnulinux-x86_64.tar.gz (Alternate link)
  • Slackware (32-bit): starfighter-1.7-slackware-i586.tgz (Alternate link)
  • Slackware (64-bit): starfighter-1.7-slackware-x86_64.tgz (Alternate link)
  • Source code (tar.gz): starfighter-1.7-src.tar.gz (Alternate link)
  • Source code (zip): starfighter-1.7-src.zip (Alternate link)
